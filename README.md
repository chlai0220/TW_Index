這程式主要的功能分成三個部分

部分一
這裡是指 在台灣證券交易所 以及 期交所 做爬蟲的動作
主要抓取的資料有
A. 台灣加權指數
B. 台指期資料(近遠月)
C. 選擇權資料(一版 / 無夜盤)

部分二
這裡主要的工作是
將部分一的資料整理成 EXCEL 的部分
並存在Daily_ReportT這個目錄中
這部分我移除了我自己在使用的幾個FUNCTION
只保留幾個比較基礎的報表供各位研究程式碼

部分三
最後的這部分
是指將報表使用GMAIL將報表寄出
這裡我移除了 帳號 以及 密碼 還 有收件人
建議可以使用讀取檔案的方式會比較安全
只是要稍加修改
